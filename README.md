"# MixologyMAUIApp" 

## Описание:
Кроссплатформенное мобильное приложение для любителей мороженого, где пользователи могут создавать, делиться и оценивать миксы разных вкусов мороженого. Предлагаю удобный интерфейс для поиска миксов, можно сохранять любимые вкусы и делиться с сообществом.

## Стек:
Проект включает в себя серверную часть, которая была реализована по принципам чистой архитектуры:
- .NET 9
- Web API
- EF Core 9
- PostgreSQL
- MediatR (CQS-паттерн)
- Swagger

И часть мобильного приложения:
- MAUI
- ReactiveUI

## Функционал
Функциональность сервера на текущий момент:
- Авторизация и регистрация (JWT Token и хеширование пароля PBKDF2)
- Сохранение изображений в бинарном виде в базе данных для миксов, брендов и коллекций
- Создание миксов с автогенерацией названия/описания
- Просмотр миксов с информацией об авторе, рейтинге и составе
- Фильтрация и сортировка миксов
- Работа с пользовательскими коллекциями
- Избранное
- Рейтинги
- Просмотр брендов и вкусов мороженого
- Поиск по брендам и вкусам
- Пагинация миксов

Функциональность клиента на текущий момент:
- Страница регистрации (http)
- Страница входа (http)
- Страница с отображением миксов с поиском и фильтрацией (http)
- Страница с отображением брендов с поиском (http)
- Страница лаборатория (создание миксов) (http)

Доп. функционал:
- Интеграция Firebase - получение уведомлений через конкретный канал
